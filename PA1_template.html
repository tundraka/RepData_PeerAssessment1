What is the average daily activity pattern?

We want to know on average how many steps are given on a time period. Remember, the data set contains a variable called interval which is the 5 minute period where the read was taken, for example, 0 represents 0:00, 1455 represents 14:55, and so on.

In this case we calcuated the average steps grouped by the interval across the entire data set. The plot displays the mean steps for interval 1000; to calculate this, all existing steps in the 1000 interval were averaged.

meanByInterval <- data[,.(avgSteps=mean(steps, na.rm=T)), .(interval)]
ggplot(meanByInterval, aes(interval, avgSteps)) +
    geom_line() +
    labs(title='Average steps by interval', y='Steps')

We can see that the interval with the max average steps is the 835 interval with an average of 206.1698113.

Imputing missing values

This data set contains a total of 2304 NA values in the steps variables. In order to fix this we will replace those NAs with the median of the day. In the days where the median cannot be calculated (there are no reading for that specific day) we will set the steps to 0.

Going from NAs to 0 is probably not the optimal. One improvement for this would be to calculate the mean of the observations of other days with observation, I'd assume that a similar routine is followed for example in every Monday.

The steps to replace NAs with the media is as follow.

  1. Replicate our data set.
  2. Detect the days that have NAs in it.
  3. Apply the function replaceNAs which will do the following
    1. It will get the median using the mean/median set displayed above.
    2. If the row doesn't exist we default to a median 0
    3. The selected median is applied to all NAs steps.
replaceNAs <- function(dayDate) {
    dayMedian <- 0
    steps <- stepsByDay[date==dayDate]

    # If we don't have a row, then  day median is NA (there are no steps recorded
    # in the whole day, we set the rowMedian to 0
    if (nrow(steps) > 0) {
        dayMedian <- steps$medianDay
    }

    imputed[date==dayDate & is.na(steps), steps:=dayMedian]
}

# We will create a copy of the data
imputed <- data.table(data)

# We detect the dates with NAs in it
nasDates <- imputed[is.na(steps), unique(date)]

a <- lapply(nasDates, replaceNAs)
